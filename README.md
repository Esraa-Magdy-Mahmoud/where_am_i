# where_am_i
Two mobile robots models were built and the robots became able to navigate and reach a specified goal while at the same time localize themselves using AMCL. 


## Requirements:
  * Ubuntu 16.04.
  * ROS Kinetic.
  * gazebo 7.7.0.
